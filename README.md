py2048
======

A simple implement of the game 2048 using python 


Needed libraries
------------------------------------
> * numpy
> * msvcrt

How to play?
-----------------------------------
> * open cmd.exe and cd to the folder
> * python py2048.py
> * use a-s-d-w to simulate the sweep
